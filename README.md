# PacketAssist: SIP Traffic Flow Analyzer

PacketAssist is a high-performance, premium web application designed for deep packet inspection and correlation of SIP (Session Initiation Protocol) traffic. It provides interactive ladder diagrams, security auditing, and media stream analysis.

## 🚀 Getting Started

### 📤 Uploading Captures
To begin analysis, simply **drag and drop** your capture files into the central analysis zone or click **"Select File"** to browse your local system.

### 📄 Supported Formats
PacketAssist supports the following network capture formats:
*   **.pcap** (Standard Packet Capture)
*   **.pcapng** (Next-Generation Packet Capture)
*   **.cap** (Legacy Capture formats)

## 🛠️ Features
*   **Interactive Ladder Diagrams**: Visualize SIP flows between endpoints.
*   **Security Detector**: Automated detection of DDoS, scanning, and malformed packet attacks.
*   **Troubleshooting Engine**: Real-time health scoring and issue identification.
*   **Media Analysis**: Map SDP negotiations and verify RTP stream parameters.

## 💻 Tech Stack
*   **Frontend**: React + TypeScript + Vite
*   **Styling**: Vanilla CSS (Modern Aesthetics)
*   **Charts**: D3.js
*   **State Management**: Zustand
